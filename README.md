My Laptop Requirements
======================

Minimum Hardware
----------------
- CPU:                    Core i5 or i7
- RAM:                    8GB or 16GB
- Screen size:            13", 14", 15.4"
- Graphics:               2GB RAM
- HDD:                    250GB SSD or 80GB SSD + 1TB
- Optical Drive:          Should NOT be present !Important

Software
--------
- OS:                     Ubuntu or any Linux OS
